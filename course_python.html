Python Fundamentals

Variables & Data Types int, float, str, bool print() & input() Type Conversion Comments & Indentation String Methods

What you'll learn: Start with zero experience and write your first Python program in minutes. You'll understand variables (containers that store data), the four core data types — int (whole numbers), float (decimals), str (text), bool (True/False) — and how to convert between them using int(), str(), float(). You'll use print() to display output and input() to take user input. Indentation is Python's way of structuring code — getting this right from day one builds good habits. You'll also explore powerful string methods like .upper(), .strip(), .replace(), and .split().

2

Operators & Expressions

Arithmetic Operators Comparison Operators Logical Operators Assignment Operators Membership Operators (in, not in) Operator Precedence

What you'll learn: Operators are the building blocks of any expression in Python. Arithmetic operators (+, -, *, /, //, %, **) handle maths — note that // is floor division and ** is power. Comparison operators (==, !=, >, <, >=, <=) return True or False and are used in conditions. Logical operators (and, or, not) combine conditions. Membership operators (in, not in) check if a value exists in a list or string — very commonly used. You'll also learn operator precedence so your expressions evaluate in the correct order.

3

Control Flow — Conditions & Loops

if / elif / else Nested Conditions for Loop while Loop range() break / continue / pass

What you'll learn: Control flow is how your program makes decisions and repeats actions. if / elif / else lets you run different code based on conditions. Nested conditions handle multi-level logic (e.g. check age, then check membership). The for loop iterates over a list or range — e.g. for i in range(1, 11) runs 10 times. The while loop repeats as long as a condition is True — always make sure it eventually becomes False or you'll create an infinite loop! break exits a loop early, continue skips the current iteration, and pass is a placeholder when no action is needed yet.

4

Functions

def keyword Parameters & Arguments return Statement Default Arguments *args & **kwargs Lambda Functions Scope (local vs global)

What you'll learn: Functions let you write reusable blocks of code — define once, use anywhere. You define a function with def function_name(): and call it by name. Parameters are inputs the function accepts; return sends a value back to the caller. Default arguments have pre-set values if none is provided. *args accepts any number of positional arguments, **kwargs accepts any number of keyword arguments — both are very common in real-world Python code. Lambda functions are one-line anonymous functions — e.g. double = lambda x: x * 2. You'll also learn variable scope: local variables exist only inside a function, global ones are accessible everywhere.

5

Data Structures

Lists Tuples Dictionaries Sets Indexing & Slicing List Methods List Comprehensions Dictionary Methods

What you'll learn: Python's built-in data structures are essential for storing and organising data. Lists are ordered and changeable — e.g. marks = [85, 90, 78]. Tuples are like lists but immutable (can't be changed after creation) — use them for fixed data like coordinates. Dictionaries store key-value pairs — e.g. student = {'name': 'Rahul', 'age': 22} — and are extremely common in real code and APIs. Sets store unique values with no duplicates. You'll master indexing (access by position), slicing (extract a portion), and powerful list methods like .append(), .sort(), .pop(). List comprehensions let you create lists in one line — e.g. [x**2 for x in range(10)].

6

Object-Oriented Programming (OOP)

Classes & Objects __init__ Method Instance Variables & Methods Inheritance Encapsulation self Keyword

What you'll learn: OOP is the way professional Python developers structure large programs. A class is a blueprint — an object is a specific instance of that blueprint (e.g. class = Car, object = your specific car). The __init__ method runs automatically when an object is created and sets its initial values. self refers to the current object. Inheritance lets a child class reuse all the code of a parent class and add its own on top — this is how code reuse is achieved in real projects. Encapsulation hides internal details, exposing only what's needed. OOP is heavily tested in Python interviews.

7

File Handling

open() & close() read() / readline() / readlines() write() & append() with Statement Working with CSV Files os Module Basics

What you'll learn: Python can read and write files directly — critical for any data task. open('file.txt', 'r') opens a file for reading; 'w' writes (overwrites), 'a' appends. read() gets the entire file content, readlines() returns each line as a list item. Always use the with statement (e.g. with open('file.txt') as f:) — it automatically closes the file even if an error occurs, which is the professional way to handle files. You'll also read and write CSV files using Python's built-in csv module, and use the os module to check if a file exists, create directories, or list files in a folder.

8

Exception Handling

try / except Multiple except Blocks else & finally raise Common Exceptions

What you'll learn: Real programs encounter errors — file not found, division by zero, invalid user input. Exception handling lets your program deal with these gracefully instead of crashing. try wraps the risky code, except catches the error and handles it. You can have multiple except blocks for different error types — e.g. except FileNotFoundError, except ValueError. finally runs regardless of whether an error occurred — perfect for closing files or database connections. raise lets you throw your own custom error. Common exceptions you'll practise: ZeroDivisionError, TypeError, IndexError, KeyError.

9

Python for Data Analytics — NumPy & Pandas

NumPy Arrays Array Operations Pandas Series & DataFrame read_csv() & read_excel() head(), tail(), info(), describe() Filtering & Sorting groupby() & agg() Handling Missing Data

What you'll learn: This is where Python becomes a data superpower. NumPy provides fast multi-dimensional arrays and mathematical operations — the backbone of all data science libraries. Pandas is the go-to tool for data analysis in Python. You'll load datasets from CSV and Excel using pd.read_csv(), explore them with .head(), .info(), and .describe(), and clean them by handling missing values with .fillna() and .dropna(). You'll filter rows, sort by columns, and use .groupby() to summarise data by category — the Pandas equivalent of SQL's GROUP BY. This module is the most important for any data analyst role.

10

Data Visualisation — Matplotlib & Seaborn

Line Chart Bar Chart Histogram Scatter Plot Pie Chart Heatmap (Seaborn) Labels, Titles & Legends

What you'll learn: Numbers alone don't tell a story — charts do. Matplotlib is Python's foundational plotting library. You'll create Line Charts (trends), Bar Charts (comparisons), Histograms (distribution of data), and Scatter Plots (relationships between two variables). plt.title(), plt.xlabel(), plt.legend() make your charts presentation-ready. Seaborn is built on top of Matplotlib and makes beautiful statistical charts with less code — particularly the Heatmap, which is essential for showing correlations in a dataset. Knowing how to visualise data is a core skill in every data analytics job.

Automation Scripts

Web Scraping (requests + BeautifulSoup) Working with APIs Automating Excel (openpyxl) Sending Emails (smtplib) Scheduling Tasks

What you'll learn: Python's true power is automation — making your computer do repetitive tasks for you. Web Scraping with requests and BeautifulSoup lets you extract data from any website automatically. API integration connects your Python code to live services (weather, stocks, news) using requests.get() and parsing JSON responses. Excel automation with openpyxl lets you read, write, and format Excel files without opening Excel. Email automation with smtplib lets you send emails programmatically — combine it with Pandas to auto-generate and email daily reports.
