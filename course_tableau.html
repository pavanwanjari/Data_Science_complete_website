Tableau Setup & Introduction

Download & Install Tableau Public Tableau Interface Overview Connecting to Data Sources Data Pane & Dimensions vs Measures Tableau vs Excel

What you'll learn: Get Tableau Public installed (it's free!) and take a guided tour of the workspace — Sheets, Data Pane, Rows/Columns shelves, and the Marks card. You'll connect to Excel and CSV files, then learn the most important concept in Tableau: the difference between Dimensions (categories like Region, Product) and Measures (numbers like Sales, Profit). By the end of this module you'll have created your very first chart from a real dataset.

2

Basic Chart Building

Bar Chart Line Chart Pie Chart Area Chart Scatter Plot Dual Axis Chart Show Me Panel

What you'll learn: Build the most commonly used chart types from scratch. You'll learn when to use each chart — Bar Charts for comparisons, Line Charts for trends over time, Scatter Plots to find correlations between two measures, and Dual Axis Charts to overlay two different metrics on the same view. The Show Me panel lets you instantly switch between chart types with one click, helping you explore your data faster.

3

Filters & Sorting

Dimension Filters Measure Filters Date Filters Top N Filter Quick Filters Sorting Data Filter Order of Operations

What you'll learn: Control exactly what data your audience sees. Filter by category (e.g. show only "Electronics"), by a number range (e.g. Sales > 10,000), or by a specific date window. The Top N Filter is especially useful in interviews — it lets you show only your top 5 or 10 performers dynamically. You'll also learn Tableau's Filter Order of Operations, which determines the sequence in which filters are applied — a critical concept that many analysts get wrong.

4

Calculated Fields

Creating Calculated Fields String Functions Date Functions Numeric Functions Logical Functions (IF / CASE) Aggregate Functions Table Calculations

What you'll learn: Calculated Fields let you create new data columns on the fly — without touching the original dataset. You'll write IF/CASE logic to classify data (e.g. "High / Medium / Low" based on sales), use Date Functions like DATEDIFF and DATEPART to calculate time between events, and clean text with String Functions like TRIM and UPPER. Table Calculations like Running Total, % of Total, and Rank are among the most asked-about topics in Tableau interviews.

5

Advanced Charts & Maps

Tree Map Heat Map Bullet Chart Waterfall Chart Box & Whisker Plot Geographic Map Filled Map

What you'll learn: Go beyond basic charts to communicate complex insights clearly. Tree Maps show part-to-whole relationships using area size. Heat Maps use color intensity to reveal patterns across two dimensions. Waterfall Charts are perfect for showing cumulative changes (e.g. how monthly profits lead to a yearly total). Geographic Maps plot data on a real map by country, state, or city — ideal for any regional sales or performance analysis.

6

Dashboard Design & Actions

Building a Dashboard Layout Containers Filter Actions Highlight Actions URL Actions Navigation Buttons Dashboard Formatting

What you'll learn: A dashboard combines multiple sheets into one view. You'll use Layout Containers (horizontal and vertical) to precisely position your charts. The real power comes from Actions — clicking a bar in one chart can automatically filter all other charts on the dashboard (Filter Action), highlight related rows (Highlight Action), or open a web page (URL Action). You'll also learn formatting tips to make your dashboards look clean and professional.

7

Parameters & Sets

Creating Parameters Parameters in Calculated Fields Dynamic Reference Lines Creating Sets Combined Sets Set Actions

What you'll learn: Parameters are user-controlled inputs — for example, a slider that lets the viewer choose a sales target, which then dynamically updates a reference line on the chart. Sets let you define a custom group of data points (e.g. "Top 10 Customers") and compare them against everything else. Combined with Set Actions, clicking on a chart element can dynamically add or remove it from a Set — creating a very engaging interactive experience.

8

LOD Expressions

What is LOD? FIXED INCLUDE EXCLUDE LOD vs Table Calculations Practical LOD Use Cases

What you'll learn: LOD (Level of Detail) expressions are one of the most powerful — and most asked-about — features in Tableau. They let you compute aggregations at a different granularity than what's in the current view. For example, FIXED calculates the result independently of the view (e.g. total sales per customer regardless of which region is filtered). INCLUDE adds more granularity, and EXCLUDE removes a dimension from the calculation. Mastering LOD separates junior analysts from senior ones.

9

Data Blending & Joins

Joins in Tableau Inner / Left / Right Join Data Blending Unions Relationships Cross-Database Joins

What you'll learn: Real-world data rarely lives in one table. Joins let you combine two tables from the same data source by a common key (e.g. customer ID). Data Blending is used when your data lives in two different sources (e.g. one sheet in Excel and one in Google Sheets). Unions stack tables that have the same columns — useful when you have monthly data in separate files. Relationships are Tableau's modern approach to multi-table analysis, letting Tableau handle the join logic automatically based on context.

10

Storytelling with Data

Creating Story Points Narrative Design Annotations & Callouts Reference Lines & Bands Trend Lines Forecasting

What you'll learn: Data is only valuable if people understand it. Story Points let you chain dashboards and charts into a guided narrative — like a presentation, but interactive. You'll add Annotations to highlight key data points (e.g. "Sales dropped here due to supply issues"), draw Reference Lines to show targets or averages, and use Trend Lines to show the direction of data over time. Forecasting uses Tableau's built-in exponential smoothing model to predict future values — a favourite in executive dashboards.

Real Dataset Project Interactive Dashboard Build Publishing to Tableau Public Top Interview Questions Scenario-based Questions Performance Optimization Tips
