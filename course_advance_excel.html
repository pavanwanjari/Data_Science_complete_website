Excel Basics Refresh

Workbook & Worksheet Navigation Cell References — Relative, Absolute ($), Mixed Number Formatting Keyboard Shortcuts Named Ranges Paste Special Freeze Panes & Split View

What you'll learn: Before jumping into advanced features, you'll nail the fundamentals that most people never truly master. The most important concept here is cell references: Relative references (like A1) shift automatically when you copy a formula — so =A1+B1 becomes =A2+B2 when pasted down. Absolute references (like $A$1) are locked and never change — use this when a formula always needs to refer to a fixed cell like a tax rate or target value. Mixed references lock only the row (A$1) or only the column ($A1). Getting this right eliminates 90% of formula errors. You'll also learn the most valuable keyboard shortcuts that save hours every week: Ctrl+Shift+End to jump to the last used cell, Ctrl+D to fill down, Alt+E+S for Paste Special. Named Ranges let you refer to a range by name — e.g. write =SUM(Sales) instead of =SUM(B2:B200) — making formulas readable and easier to maintain.

Lookup & Reference Formulas

VLOOKUP HLOOKUP INDEX + MATCH XLOOKUP (Excel 365) CHOOSE INDIRECT OFFSET

What you'll learn: Lookup formulas are the most tested Excel skill in every analyst interview. VLOOKUP searches a value in the first column of a table and returns a value from another column — e.g. look up Employee ID and return their Department. Common traps: the lookup column must be the leftmost column, and always use FALSE as the 4th argument for exact match. INDEX + MATCH is the professional upgrade — it can look left, right, and doesn't break when you insert columns. =INDEX(result_column, MATCH(lookup_value, lookup_column, 0)) — learn this and you'll always be the smartest person in the room. XLOOKUP (available in Excel 365) replaces both VLOOKUP and HLOOKUP with a cleaner, more powerful syntax. INDIRECT converts a text string into a cell reference — powerful for dynamic dashboards where the sheet name changes. OFFSET returns a range that shifts by a specified number of rows and columns — used in dynamic charts.

Logical & Conditional Formulas

IF Nested IF IFS (Excel 2019+) SUMIF & SUMIFS COUNTIF & COUNTIFS AVERAGEIF & AVERAGEIFS AND, OR, NOT IFERROR & IFNA

What you'll learn: These formulas add intelligence to your spreadsheets. IF is the most used formula in Excel: =IF(score>=60, "Pass", "Fail"). Nested IF handles multiple conditions: =IF(score>=90,"A", IF(score>=75,"B","C")) — but with more than 3 levels, use IFS instead, which is much cleaner. SUMIF adds only the rows that meet a condition — e.g. total sales for a specific region: =SUMIF(region_col,"North",sales_col). SUMIFS supports multiple conditions. COUNTIF counts matching rows — e.g. how many employees are in "HR"? COUNTIFS adds multiple criteria. These are used in almost every MIS report and dashboard. IFERROR silences error messages — wrap any formula with =IFERROR(your_formula, "Not Found") so your reports look clean even when lookups fail.

Date, Time & Text Functions

TODAY() & NOW() DATE, YEAR, MONTH, DAY DATEDIF — Age & Tenure EOMONTH & WORKDAY TEXT() — Format as String LEFT, RIGHT, MID LEN, FIND, SEARCH TRIM, CLEAN, PROPER CONCATENATE & TEXTJOIN

What you'll learn: Date and text operations are used in every HR report, attendance tracker, and MIS file. Date functions: TODAY() returns today's date dynamically — use =TODAY()-B2 to calculate how many days since an order. DATEDIF(start, end, "Y") calculates the age or tenure of an employee in years. EOMONTH returns the last day of the month — essential for finance reports. WORKDAY adds business days (skipping weekends) — used in project timelines. Text functions: LEFT(cell, n), RIGHT(cell, n), MID(cell, start, n) extract specific characters — e.g. extract the year from "INV-2024-001". TRIM() removes extra spaces from imported data (a lifesaver for cleaning messy reports). PROPER() fixes inconsistent capitalisation. TEXTJOIN joins multiple cells with a separator — far better than the old concatenate approach.

Data Cleaning & Validation

Remove Duplicates Find & Replace (Wildcards) Text to Columns Flash Fill Data Validation — Dropdown Lists Data Validation — Custom Rules Conditional Formatting Go To Special — Blanks & Errors

What you'll learn: Raw data from systems is always messy — cleaning it efficiently is what separates junior and senior analysts. Remove Duplicates is a one-click tool to eliminate repeated rows — but you'll learn how to identify duplicates with COUNTIF first so you can review before deleting. Text to Columns splits a single column into multiple — e.g. split "Rahul, 25, Mumbai" into three separate columns. Flash Fill (Ctrl+E) is Excel's AI — show it a pattern once and it fills the rest automatically. Data Validation creates dropdown lists so users can only select valid values — critical for any shared MIS file. Custom validation rules prevent invalid entries like negative salaries or future dates. Conditional Formatting highlights cells automatically based on rules — e.g. colour all sales below target in red, top 10 values in green, duplicate values in yellow. Go To Special selects only blank cells, only formulas, or only constants in one step — essential for bulk editing large datasets.

Pivot Tables & Pivot Charts

Creating Pivot Tables Rows, Columns, Values, Filters Calculated Fields Grouping — Dates & Numbers Show Values As — % of Total, Running Total Slicers & Timelines Pivot Charts Refreshing & Data Source Update

What you'll learn: Pivot Tables are the most powerful feature in Excel — and the most asked-about topic in analyst interviews. A Pivot Table can summarise 100,000 rows of data in seconds with zero formulas. You drag fields into Rows, Columns, Values, and Filters — and Excel instantly produces a summary. You'll learn to group dates by month, quarter, or year — so a daily sales table becomes a monthly summary in two clicks. Calculated Fields let you add custom formulas inside the pivot — e.g. calculate profit margin as % without modifying the source data. Show Values As lets you display numbers as % of row total, % of column total, or running totals — essential for performance dashboards. Slicers are visual filter buttons that make your report interactive — click "North" and every pivot on the page updates instantly. Timelines are slicers specifically for date fields. Connect multiple pivot tables to one slicer and build a fully interactive management dashboard.

Power Query — Automate Data Import & Transformation

Connect to CSV, Excel, Web Remove Columns & Filter Rows Split Column & Replace Values Merge Queries — JOIN Tables Append Queries — Stack Tables Group By — Summarise Data Unpivot Columns Refresh All — One-Click Update

What you'll learn: Power Query is the biggest productivity upgrade in Excel in 20 years — and most Excel users don't know it exists. Instead of manually cleaning data every month, you build a transformation pipeline once and click Refresh All every month — Power Query re-applies every step automatically. Connect to CSV files, Excel workbooks, folders full of files, or websites as your data source. Transform the data: remove unwanted columns, filter rows, split columns, replace values, and change data types. Merge Queries is Power Query's version of VLOOKUP — join two tables by a common column (like a SQL JOIN) without any formula. Append Queries stacks multiple tables vertically — e.g. combine 12 monthly sales files into one table instantly. Unpivot converts wide (crosstab) tables into long (flat) tables — essential for feeding data into Pivot Tables or Power BI. Power Query alone can save 5–10 hours of manual work per week.

Macros & VBA Basics — Automate Repetitive Tasks

Record Macro Run & Assign Macro to Button VBA Editor (Alt+F11) Sub & End Sub Variables & MsgBox Loops — For Each Row If Condition in VBA Automate Formatting & Copy-Paste

What you'll learn: Macros let you record a sequence of actions in Excel and replay them instantly with one click or button press. Record Macro captures everything you do — formatting, filtering, copying, pasting — and plays it back every time without you repeating the steps. You'll then open the VBA Editor (Alt+F11) to see the auto-generated code and make small edits to make it smarter. You'll write basic VBA: Sub defines a macro, MsgBox shows a pop-up message, variables store values, For Each loops process every row in a table. You'll use VBA to automate formatting (apply colour based on value), automate copy-paste (copy filtered data to a new sheet), and create a one-click report generator. You don't need to become a programmer — just enough VBA to automate your most repetitive Excel tasks saves you hours every week.
